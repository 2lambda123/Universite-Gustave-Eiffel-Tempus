Detailed Description

template<class OGraph>
class Tempus::ReverseGraphT< OGraph >

Reverse graph adaptor (read only) for Road and Public Transport This is inspired by boost::reverse_graph But this one does not create a new type for vertex / edge descriptors for the reverse graph which simplifies things
