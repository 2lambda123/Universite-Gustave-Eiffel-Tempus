Vertices are represented by their CH ordering Edges are organized in the following way: pairs of vertices are sorted in ascending order by the first element of the pair. All upward edges are grouped together first and then all downward edges up_it points to a sequence representing the number of upward edges for each vertex

Example: if vp points to the following sequence : (0,1)(0,2)(0,1)(1,2)(1,2) if up_it points to the following sequence : (2,1,0) with n_vertices = 3 then it corresponds to the following edges: 0->1, 0->2, 1->0, 1->2, 2->1
