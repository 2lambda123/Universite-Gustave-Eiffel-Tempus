Detailed Description

A Roadmap is an object used to model steps involved in a multimodal route. It is a base for result values of a request.

Some of the fields may stay empty and will be filled by a request on the auxiliary storage (db), since there is no need for them to be always present in memory (geometries, names, etc.). They are marked as "retrieved from db"
